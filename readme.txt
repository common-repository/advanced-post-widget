Builds post widget based on options you choose from a form in a widget

== Description ==
Builds post widget based on options you choose from a form in a widget. It works similar to SQL query but no coding knowledge is required.
You can drag & drop the widget in as many sidebars or custom widget areas as you like and build as many independent lists as you like. You can easily build any type of widget youâ€™d like.
You can create a fully functional website using this widget.



= Links =
* [Visit my site](https://mrkindy.com).

= Customize front-end =

By default the widget looks like all other widgets from the same sidebar you drop it in. It has its own CSS file that overrides the default styles for your list elements. You can use that file to further customize the front-end.
You can also add custom template.

Upload the Pray Time plugin to your blog, Activate it, then configure your widget from theme wigets.
